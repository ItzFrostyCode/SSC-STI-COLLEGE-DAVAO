# SSC-STI College Davao Portal

**System Developer**: JOSHUA WAYMAN A. ARABEJO  
**Academic Year**: 2025-2026  

## Project Overview
This project is a centralized web portal and announcement system designed for the Supreme Student Council (SSC) of STI College Davao. It was developed as an academic project to streamline student communication, manage events, showcase officers, and provide a modern, interactive feed for campus announcements.
